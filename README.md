# Android Telecom Monitoring App

Android-based LTE/5G telecom and device monitoring application with:
- LTE/5G measurement collection
- Neighbor-cell monitoring
- GPS/location tracking
- Background monitoring using WorkManager
- Battery/RAM/storage monitoring
- CSV dataset generation
- Live signal graph visualization

## Features

### Telecom Monitoring
- 2G/3G/4G/5G detection
- LTE information
- 5G NR information
- RSRP / RSRQ / SINR
- PCI / TAC / NCI
- Neighbor-cell collection
- Signal strength monitoring

### Device Monitoring
- Battery percentage
- Battery temperature
- Battery voltage
- RAM monitoring
- Storage monitoring

### Background Monitoring
- WorkManager integration
- Periodic background tasks
- Background CSV logging

### Visualization
- Live dBm graph
- Signal-strength visualization

### Logging
- CSV dataset generation
- Telecom logs
- Background device logs

## Technologies Used
- Kotlin
- Android SDK
- WorkManager
- MPAndroidChart
- Android Studio
